# Template: New Skill

Create a `.md` file in the agent's `skills/` directory, or in the root `skills/` directory for shared skills.

## Format

```markdown
# Skill: [Name]

## Purpose
[One line: what this skill accomplishes]

## Steps
1. [Step 1]
2. [Step 2 — can reference other skills by path]
3. [Step 3]

## Output
[What this skill produces]
```

Keep it concise. A skill is a repeatable SOP, not a manual.
