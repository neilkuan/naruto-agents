# Skill: Morning Briefing

## Purpose
Generate a daily startup overview for the human.

## Steps
1. Read `memory/daily/` for yesterday's team summary (if exists)
2. Read `memory/MEMORY.md` for ongoing priorities
3. Check each agent's `memory/daily/` for recent activity
4. Compile: what was done yesterday, what's pending, what needs attention today

## Output
A concise daily briefing in markdown.
